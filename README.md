# Iris EDA

Exploratory data analysis on the Iris dataset using Python, Pandas, Matplotlib, and Seaborn.

## Aim
- Understand the characteristics of the Iris dataset.
- Explore relationships between sepal/petal length and width for each species.

## Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- scikit-learn

## What I Did
- Loaded the Iris dataset into a Pandas DataFrame.
- Analyzed basic statistics and class balance.
- Created visualizations (histograms, boxplots, pairplots, correlation heatmap) to study feature distributions and relationships.
